Common CSS Properties

Property Name Description
<font-family> Font Family Used to specify the typeface.
<font-size> Font Size Used to set text size.
<font-weight> Font Weight Used to set weight of text such as bolded or normal.
<font-align> Font Align Used to set text alignment such as left, center or right.
<color> Foreground Color Used to set the foreground or text color.
<background-color> Background Color Used to set the background color.
<opacity> Opacity Used to set opacity of an element.
